### Prerequisites

Make sure you have installed:

* Flutter SDK (latest stable version)
* A code editor (e.g., VS Code, Android Studio)
* Optional: device/emulator for mobile testing OR browser for Web
* Optional: configure desktop targets if you want Windows/macOS/Linux support

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/Chris-Eminence/cryptx-app.git
cd cryptx-app
```

2. **Install dependencies**

```bash
flutter pub get
```

3. **(Optional) If you're targeting mobile, run:**

```bash
flutter run
```


4. Explore the code in `lib/` .

### Development Notes

* The main codebase lives in `lib/`.
* You'll find assets (images/icons) under `assets/png/`.
* Platform-specific configuration is under the respective platform folders (e.g., `android/`, `ios/`).
* To add new features, consider using a modular folder structure for maintainability (for example: `ui/`, `widgets/`, `models/`, `services/`).
* For state management, pick your preferred package (Provider, Riverpod, Bloc, etc) and plug into the architecture.
